# fetch-backend-oa

## Prerequisites
- Docker installed

## Clone the repository
```code
git clone https://github.com/hassanuahmad/fetch-backend-oa.git
cd fetch-backend-oa
```

## Build and Run

1. Build the Docker image:
```bash
docker build -t fetch-challenge .
```

  2. Run the Docker container:
  ```bash
docker run -p 8080:8080 fetch-challenge
```
